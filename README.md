# Goldwire Games — Trophic

**[goldwiregames.com](https://goldwiregames.com)**

Trophic is an asymmetric engine-building card game about ecosystems. Players build trophic pyramids from producers to apex predators across distinct biomes — each with its own playstyle, resource economy, and evolutionary advantages.

## The Game

- **1–4 players** | 45–60 minutes | Ages 12+
- **4 Biome decks**: Savanna, Arctic, Amazon, Rockies
- **Core mechanic**: Activate a producer at the base of your pyramid and watch energy cascade upward through your food web
- **Constructed play**: Build custom 50-card decks using the color system — every animal has ecological colors that span across biomes
